Recommendations to Reduce Text Heaviness:

Option 1: Keep Both But Shorter

  • Shorten both description and signal to essential info only
  • Use bullet points or very concise phrasing
  • Example: "AI verification with EigenLayer" + "StarkWare partnership"

Option 2: Description Only

  • Remove signal entirely to keep cards clean
  • Can embed key partnerships in description with color highlight
  • Reduces visual clutter significantly

Option 3: Conditional Signal Display

  • Only show signal for ALPHA tier or exceptional partnerships
  • Most cards stay clean with just description
  • Special projects get the extra highlight they deserve

My recommendation: Option 2 (Description Only) or Option 3 (Conditional) to avoid text overload.
