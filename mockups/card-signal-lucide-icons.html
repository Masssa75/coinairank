Icon Implementation Guide

Best Icons for "Signal" Concept:

  • Radio (signal waves) - Most literal signal representation
  • Wifi - Universal signal strength icon
  • Activity (pulse) - Shows vital signs/important activity
  • Zap (lightning) - Power/performance signals (ALPHA tier)
  • Arrow/Chevron - Simple, clean, directional

Implementation in React/Next.js:

// Using Lucide React (already in your project)
import { Radio, Wifi, Activity, Zap, ArrowRight, ChevronRight } from 'lucide-react';

// In your component
{project.top_signal && (
  <div className="flex items-center gap-2 text-[13px] text-gray-400 mb-5 -mt-2">
    {project.tier === 'ALPHA' ? (
      <Zap className="w-3.5 h-3.5 text-[#00ff88]" />
    ) : (
      <ArrowRight className="w-3.5 h-3.5 text-[#00ff88]" />
    )}
    <span>{project.top_signal}</span>
  </div>
)}

Recommended Approach:

Use ArrowRight for general signals
Use Zap for ALPHA tier projects
